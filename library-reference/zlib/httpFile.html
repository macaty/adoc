zlib.httpFile 成员列表

自网络下载并解压 .zip、.tar.gz、*.tgz, *.gz 文件

zlib.httpFile.download(下载网址,对话框标题,下载路径,解压目录,存储文件名,父窗口)

自网络下载并解压*.zip、*.tar.gz、*.tgz 文件
,除下载网址以外，其他所有参数可选,
下载网址允许传入一个动态返回 url 的线程函数,
下载路径可指定存储目录或存储文件路径
失败返回null
成功返回解压目录

zlib.httpFile.headers

下载时添加的HTTP请求头

zlib.httpFile.referer

下载时的HTTP引用页

zlib.httpFile.userAgent

下载时用到的User Agent
