gdip.exifTags

Exif 标签列表
此名字空间存储的键 Exif 的 ID 时对应的值为标签名称，
键为标签名称时对应的值为数值类型的 ID。

创建 Exif 对象。

gdip.exifTags()

返回对象:gdpExifTagsObject

gdip.exifTags(imageOrBitmap)

创建 Exif 对象。
参数 @1 可指定 gdip.image 或 gdip.bitmap 对象。

gdip.exifTags(imagePathOrBuffer)

创建 Exif 对象。
参数 @1 可指定图像路径中内存数据。
