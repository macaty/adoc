sqlite 成员列表

sqlite.aes128

支持加密功能的 sqlite 扩展库,
导入此扩展库以后,代码中仍然可以使用 sqlite
参考: http://sourceforge.net/projects/wxcode/files/Components/wxSQLite3/

sqlite.aes128("\数据库路径")

这个函数是 sqlite 构造函数的别名,
参数一指定数据库路径,支持自动创建数据库文件.
第二个参数(可选参数)指定数据库密钥

sqlite.aes128()

返回对象:sqliteConnObject
