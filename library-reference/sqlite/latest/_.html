sqlite.latest

sqlite 扩展库别名,但引入新版 SQLite.dll 组件,
导入此扩展库以后,代码中应当继续使用 sqlite 而非 sqlite.latest

标准库 sqlite 考虑体积不会使用最新版本 SQLite.dll。
sqlite.latest 扩展库则会不定期更新 SQLite.dll,
但不会过于频繁更新，也不保证与 SQLite 官方同步。

大家可以参考此扩展库的源码引入指定版本的 SQLite.dll

sqlite.latest()

返回对象:sqliteConnObject
