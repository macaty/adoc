# string.words 库模块帮助文?
## string 成员列表

### string.words

简单英中词?
键为英文，值为中文，数据仅供参考?
### string.words(英文单词)

返回英文单词的中文翻译，如果参数?null 则直接返回?
参数@1指定的英文单词先移除首尾空格并转换为小写，然后再查询中文解释?
首个返回值为中文解释，第二个返回值为去除首尾空格并转为小写的输入文本?
如果去除首尾空格不是纯字母文本，则无任何返回值?
如果改为下标方式传入英文单词，则不会移除空格，也不会转为小写?
