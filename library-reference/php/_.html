php 成员列表

php.begin()

启动PHP解释器
此函数应当与php.end()函数成对调用

php.dofile("/")

在参数中指定文件路径
也可以增加一个参数直接传入PHP代码.

php.end()

重置PHP解释器
此函数应当begin()函数成对调用.

php.eval()

返回PHP表达式的值.
此函数在遇到PHP致命错误时、会中断进程
推荐使用php.exec()函数替代.

php.exec()

执行PHP代码.

php.print

php.print = function( msg ) {  
    io.print("PHP输出内容:", msg)   
}
