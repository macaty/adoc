java.yuiCompressor 成员列表

java.yuiCompressor.compress("源文件路径","输出路径","编码")

压缩 js 或 css 文件。
源文件路径可指定网址或本地文件。
编码为可选参数,默认为"utf-8"
如果未找到java运行时,该函数为空值。
成功返回 true ，失败返回 null,错误信息。
如果参数 @2 不指定输出文件路径，压缩成功返回压缩后的文件数据。

java.yuiCompressor.compressCss("CSS文件路径","输出路径","编码")

压缩css文件。
源文件路径可指定网址或本地文件。
编码为可选参数,默认为"utf-8"
如果未找到java运行时,该函数为空值。
如果参数 @2 不指定输出文件路径，压缩成功返回压缩后的文件数据。

java.yuiCompressor.compressJs("JS文件路径","输出路径","编码")

压缩js文件。
源文件路径可指定网址或本地文件。
编码为可选参数,默认为"utf-8"
如果未找到java运行时,该函数为空值。
如果参数 @2 不指定输出文件路径，压缩成功返回压缩后的文件数据。

java.yuiCompressor.disableOptimizations

js压缩选项,设为true禁止优化

java.yuiCompressor.lineBreak

指定列数以自动换行，默认值已指定为 1000 。
因为 ScriptControl 的限制 web.script 不支持 JS 一行的字数太长。
现在很多压缩 JS 都会因为单行太长导致 web.script 无法加载。
但 ScriptControl 控件加载每行最大字数不超过 1000-2000（因版本不同） 是安全的。

java.yuiCompressor.nomunge

js压缩选项,设为true仅压缩不混淆

java.yuiCompressor.preserveSem

js压缩选项,设为true保留所有分号

java.yuiCompressor.verbose

显示详细信息
