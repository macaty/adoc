java.ffdec 库模块帮助文档

java 成员列表

java.ffdec()

返回对象:processPopenObject

java.ffdec(...)

启动 SWF 反编译工具 JPEXS Free Flash Decompiler,
可选指定一个或多个启动参数,
可用参数:
https://github.com/jindrapetrik/jpexs-decompiler/wiki/Commandline-arguments
需要 JRE 1.8 或 1.8 以上版本,
aardio 查找JRE的顺序请参考 aardio 调用 JAVA 范例
支持系统安装的JRE或者用环境变量 JRE_HOME 指定 JRE 路径
