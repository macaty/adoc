System.Windows.Automation.3 库模块帮助文档

说明

导入 System.Windows.Automation.3 以后， 调用时仍然使用 System.Windows.Automation，用法基本兼容。 扩展了一些原来需要通过 COM 接口调用的功能，例如支持 TextPattern2 。

System.Windows.Automation.3 调用开源组件 UIAComWrapper： https://github.com/TestStack/UIAComWrapper/

UIAComWrapper 开源许可证： https://github.com/TestStack/UIAComWrapper/blob/master/LICENSE.txt

不需要再导入 System.Windows.Automation 。
