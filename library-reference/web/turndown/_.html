aardio 文档

web.turndown 库模块帮助文档

web 成员列表

web.turndown()

创建 TurndownService 对象，用于将 HTML 转换为 Markdown。
参数 @1 可用 aardio 表对象指定选项，
也可用字符串参数指定包含描述 JS 选项对象的 JS 代码。

返回对象:WebTurndownObject

WebTurndownObject 成员列表

WebTurndownObject.addRule(key,jsRule)

添加规则。
参数 @key 为字符串，
参数 @jsRule 用包含 JS 代码的字符串指定规则。
注意 JS 必须兼容 IE 内核，不要使用箭头函数这些。

WebTurndownObject.turndown()

将参数 @1 指定的 HTML 代码转换为 Markdown 文本。

WebTurndownObject.turndownFile()

将参数 @1 指定的 HTML 代码文件转换为 Markdown 文本。

WebTurndownObject.turndownUrl()

将参数 @1 指定网址的 HTML 代码文件转换为 Markdown 文本。
可选用参数 @2 指定预处理 HTML 代码的回调函数，该回调函数接受一个参数以获取 HTML 并返回新的 HTML。
如果之前已导入 web.view 则使用 web.view 打开网址并获取 HTML。

WebTurndownObject.useGfm()

启用 gfm（GitHub Flavored Markdown） 插件，
可选用一个或多个字符串参数指定要启用的插件名称。
