## web 成员列表

### web.joinHeaders(HTTP?...)

拼接 HTTP ?支持任意个参?

HTTP 头可以是字符串、表（数组、键值对）?
字符串参数总是移除最前面的回车换行，并在尾部添加回车换行?
单个字符串参数如果包含多?HTTP 头应当以回车换行分隔?
如果表参数是键值对,则值包含的宽字符进?UrlEncode 编码（UTF8）?
键值对的值也可以是多个值组成的数组?
如果表参数是数组，则使用回车换行作为分隔符拼接为字符串?
返回拼接后的字符串，HTTP 头之间以回车换行分隔，尾部总是添加回车换行
