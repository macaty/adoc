process 成员列表

process.evb

Enigma Virtual Box 独立 EXE 封包工具
生成的独立 EXE 默认不会在运行时释放临时文件

仅支持 32 位程序，现在知道 32 位程序香了吧！

process.evb()

返回对象:processPopenObject

process.evb(evbPath)

如果用 @evbPath 指定 evb 工程文件路径，
则创建并返回进用于生成独立 EXE 并返回 process.popen 对象。
否则打开 Enigma Virtual Box 图形界面
