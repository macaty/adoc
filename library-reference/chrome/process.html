Chrome 启动参数参考

chrome.process()

返回对象:processObject

chrome.process(启动参数表,优先启动Chrome)

启动 Edge（Chromium ）、Chrome 、 Supermium 等 Chrome 内核浏览器。
如果参数 @2 为 true 则优先启动 Chrome，否则优先启动 Edge。
启动参数表中键名应以 -- 开始。

所有参数都可以省略。
启动成功则返回 process 对象
