julia.module 库模块帮助文档

julia 成员列表

julia.module()

返回 Julia 模块对象,
可选在参数@2指定导入模块的方法"using"或"import"

返回对象:juliaModuleObject

julia.module 成员列表

julia.module.Base

Base 模块

返回对象:juliaModuleObject

julia.module.Core

Core 模块

返回对象:juliaModuleObject

julia.module.Main

Main 是 Julia 主模块。
代码默认在此模块下运行

返回对象:juliaModuleObject

juliaModuleObject 成员列表

juliaModuleObject.?

输入模块的全局变量名或者函数名。
可以直接调用模块下的函数
