# julia.pkg 库模块帮助文?
## julia.pkg 成员列表

### julia.pkg.add()

添加参数 @1 指定的模块?
参数 @1 用一个字符串指定模块名?
如果模块无法导入则安装模块?
否则安装模块，安装成功则导入模块?
成功返回模块对象

[返回对象:juliaModuleObject](#juliaModuleObject)

### julia.pkg.setServer(\_)

设置镜像服务器?
参数 @1 用一个字符串指定镜像服务器地址
