# odisasm 库模块帮助文?
## odisasm 成员列表

### odisasm.asm(汇编代码)

32位汇编一行返回机器码

### odisasm.dasm(机器?

32位反汇编,

可返回继续的多行汇编代码

### odisasm.eachAsm(汇编代码)

```aardio aardio
for( bin,line in odisasm.eachAsm(/*汇编代码*/) ){
    io.print( odisasm.hex(bin),line )
}

```

### odisasm.eachDasm(不定个数机器?

```aardio aardio
for code in odisasm.eachDasm(
    '\x55', /*push ebp*/
    '\x89\xE5', /*mov ebp,esp*/
    ){
    io.print( code )
}

```

### odisasm.hex(二进制数?

返回十六进制编码字符?
### odisasm.nextAsm(汇编代码)

32位汇编一行返回机器码,汇编代码,下一行汇编代?

失败返回null,错误信息
