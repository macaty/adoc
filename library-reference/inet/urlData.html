inet.urlData 库模块帮助文档

inet 成员列表

inet.urlData(data,path,mime)

参数 @1 可指定字符串或 buffer，如果为 null 则自参数 @2 指定的路径加载数据。
参数 @2 指定文件路径，如果用于获取 MIME 可指定并非实际存在的文件名。
可选指定 MIME，如不指定则自参数 @2 指定的路径获取 MIME。

inet.urlData.parse

解析 Data URL 并返回解析后数据

inet.urlData.parse(url)

如果参数不是字符串返回 null 。
如果参数是 Data URL，返回解析后的数据与 MIME。\如果数据由 base64 编码则自动解码。
