win.appBar 库模块帮助文档

win 成员列表

win.appBar()

返回对象:winAppBarObject

win.appBar(句柄)

可选指定句柄,
可选在第二个参数中使用表对象指定APPBARDATA属性

win.appBar 成员列表

任务栏

win.appBar.autoHide( 任务栏句柄,false)

取消自动隐藏

win.appBar.autoHide( 任务栏句柄,true)

设置为自动隐藏

win.appBar.command()

发送命令,
参数@1指定命令 ID，参数为 419 可显示桌面。

win.appBar.find()

返回系统任务栏句柄

win.appBar.getAutoHideBar()

返回左,上,右,下四个被设置为自动隐藏的任务栏句柄

win.appBar.getTaskBar()

返回系统任务栏信息,失败返回null

返回对象:STDAPPBARDATAObject

win.appBar.isAutoHide( 任务栏句柄 )

任务栏是否自动隐藏

win.appBar.message(_ABM,{} )

发送任务栏命令,
可选在第二个参数中使用表对象指定APPBARDATA属性

win.appBar.regist(窗口句柄,自定义回调消息ID)

将窗口注册接收任务栏通知消息,例如 _ABN_FULLSCREENAPP

win.appBar.setAutoHideBar( 任务栏句柄,_ABE_BOTTOM )

设置为自动停靠

win.appBar.unRegist(窗口句柄)

取消注册的任务栏回调消息
