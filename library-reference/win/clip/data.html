win.clip.data 库模块帮助文档

win.clip 成员列表

win.clip.data

剪贴板读写自定义格式数据

win.clip.data()

创建剪贴板自定义格式。
参数 @1 用一个字符串指定格式名称。
例如 RTF 格式为 "Rich Text Format"

返回对象:winClipDataObject

winClipDataObject 成员列表

winClipDataObject.read()

读取自定义格式数据。
成功返回 buffer 对象。

winClipDataObject.readText()

读取 Unicode（UTF-16）编码文本并返回 UTF-8 编码文本。

winClipDataObject.write(剪贴板数据)

写入自定义格式数据，不清空剪贴板原来的数据。
参数@1为 buffer 或 string 对象。
成功返回剪贴板数据句柄

winClipDataObject.write(剪贴板数据,true)

写入自定义格式数据，清空剪贴板原来的数据。
参数@1为 buffer 或 string 对象。
成功返回剪贴板数据句柄

winClipDataObject.writeText(剪贴板数据)

将 UTF-8 编码文本转换为 Unicode（UTF-16）编码写入剪贴板。
不清空剪贴板原来的数据。

成功返回剪贴板数据句柄
