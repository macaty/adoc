win.clip.file 库模块帮助文档

win.clip.file 成员列表

win.clip.file.oleCopy()

清空剪贴板，然后复制文件且复制 OLE 对象。
参数 @1 指定要复制的文件路径。
win.clip.gif.write 函数调用了此函数复制文件

win.clip.file.read()

返回剪贴板文件路径数组,
第二个返回值指明复制类型,复制为"copy",剪切则为"move"

win.clip.file.write(文件路径)

将文件复制到剪贴板
可选使用参数@3指定在复制前是否清空剪贴板,默认清空

win.clip.file.write(文件路径,"copy")

将文件复制到剪贴板
参数@1可用字符串或字符串数组指定一个或多个文件路径
可选使用参数@3指定在复制前是否清空剪贴板,默认清空

win.clip.file.write(文件路径,"move")

将文件剪切到剪贴板
参数@1可用字符串或字符串数组指定一个或多个文件路径
可选使用参数@3指定在复制前是否清空剪贴板,默认清空
