win.region.hole 库模块帮助文档

win.region 成员列表

win.region.hole

用于挖空控件窗口所在区域

win.region.hole()

参数指定控件对象,
自动挖空控件窗口所在区域,
控件可指定自适应大小、固定边距等参数,
控件的父窗口建议设为无边框,
可使用 win.ui.simpleWindow 等创建自定义边框

返回对象:winRegionHoleObject

winRegionHoleObject 成员列表

winRegionHoleObject.enable()

参数可使用 true 或 false 指定是否挖空控件窗口所在区域
