# win.util.savePosition 库模块帮助文?
## win.util 成员列表

### win.util.savePosition(winform)

只能在调用winform.bindConfig函数之前使用

允许winform.bindConfig保存上次关闭时的窗口位置,并在下次打开时恢复位?

用户每次打开程序屏幕上的窗口布局都会变化,添加此功能可能会给用户带来不必要的困?

例如用户关闭时窗口是最大化?不等于用户期望下次打开也是最大化状?

建议不要滥用
