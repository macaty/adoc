win.timer 库模块帮助文档

win 成员列表

win.timer

定时管理器

win.timer()

返回对象:winTimerObject

win.timer(winform,interval)

创建定时器
如果不指定窗体对象,则创建消息窗体
interval为可选参数，用于指定间隔时间,以毫秒为单位,默认为100毫秒

winTimerObject 成员列表

winTimerObject.disable()

禁用定时器

winTimerObject.enable(interval,count)

启用定时器,
可选使用interval参数中指定间隔时间,以毫秒为单位,
可选使用count参数限定定时器执行次数,
如果指定了有效执行次数,则定时器会立即执行一次onTimer函数,
否则会先延时再执行onTimer函数

winTimerObject.getInterval()

读取定时器间隔时间

winTimerObject.onEnd

winTimerObject.onEnd = function(){
    /*定时器停止时触发此事件*/
}

winTimerObject.onTimer

winTimerObject.onTimer = function(){
    /*设置定时器触发函数,此函数不应返回任何值*/
}

winTimerObject.setInterval(1)

修改定时器间隔时间,
以毫秒为单位
