win.ui 成员列表

win.ui.simpleWindow3

简单初始化无边框窗口,并创建渐变背景的标题栏,
标题栏使用分层窗口实现，并使用 orphanWindow 悬浮在父窗口上面,
标题栏上添加最小化、最大化、关闭窗口等按钮,
并自动添加拖动边框、窗口阴影、设置窗口最大化范围等,
可以在窗体设计器的窗口属性中禁用最大化、最小化按钮
禁用最大化按钮以后不添加用于拖动调整窗体大小的边框

win.ui.simpleWindow3()

返回对象:winUiSimpleWindow3Object

win.ui.simpleWindow3(窗体对象,字体大小,按钮宽度,按钮高度,标题栏高度,背景色,前景色)

为参数@1指定的无边框窗口添加渐变背景的标题栏,
可选使用参数@2指定标题栏按钮字体大小,以像素为单位,
可选用参数@3,@4指定关闭等按钮宽度,高度
标题栏高度为可选参数,
背景色、前景色用于指定渐变顶部、底部颜色,都是可选参数
