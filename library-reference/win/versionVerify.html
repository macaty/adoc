win.versionVerify 库模块帮助文档

条件表达式语法

支持用 && 逻辑运算法连接多个条件表达式。 支持用 ==, <, <=, >, >= 等关系运算符判断版本号， 支持用 &, | 等按位运算符判断 suiteMask， 不支持其他操作符。

一般建议用 >= 或 <= , 而非 > 或 <, 用 > 或 < 可能实际检测用的是>= 或 <=。

win 成员列表

win.versionVerify("major >= 10 && minor >=0 ")

用于验证操作系统版本
参数中指定条件表达式,可支持的操作符和变量请查看源代码

建议使用标准库 win.version 获取系统版本,
或使用 aardio 提供的全局常量:
_WINXP _WIN7_LATER _WIN10_LATER _WINE
快速判断操作系统版本
