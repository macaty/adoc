win.subclass

窗口子类化:
即替换当前进程内指定窗口的消息回调函数,
用于个别无法直接指定消息回调函数的窗口

win.subclass()

返回对象:winsubclassObject

win.subclass(窗口对象, 消息回调函数)

win.subclass(  
    hwnd/*窗口对象或句柄*/,function (hwnd,message,wParam,lParam) {   

    })

winsubclassObject.free()
