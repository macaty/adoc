dotNet.audioDevice 库模块帮助文档

dotNet.audioDevice 成员列表

dotNet.audioDevice.each()

for( mmDevice,devId,devName in dotNet.audioDevice.each() ){  
    /*遍历所有音频设备,devId 为设备 ID,devName 为友好名称*/  
}

dotNet.audioDevice.getDefaultCapture()

获取默认录音设备

dotNet.audioDevice.getDefaultCaptureId()

获取默认录音设备 ID

dotNet.audioDevice.getDefaultRender()

获取默认音频输出设备

dotNet.audioDevice.getDefaultRenderId()

获取默认音频输出设备 ID

dotNet.audioDevice.setDefault()

设置默认设备,
参数@1必须指定音频设备ID,
用 dotNet.audioDevice.each 可以遍历所有音频设备 ID
