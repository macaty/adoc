dotNet 成员列表

dotNet.ilSpy

用于启动 ILSpy 反编译 .Net 程序

dotNet.ilSpy(path,...)

启动 ILSpy 反编译 .Net 程序,所有参数可省略,
参数 @path 指定 .Net 程序或程序集路径,
可添加一个或多个启动参数,

可用参数请参考:
https://github.com/icsharpcode/ILSpy/blob/master/doc/Command%20Line.txt
