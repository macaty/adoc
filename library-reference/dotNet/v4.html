dotNet.v4 库模块帮助文档

dotNet 成员列表

dotNet.v4

导入此库，如果 .Net v4.x 未安装则自动安装。
如果 .Net 4.x 已安装，则自动导入 dotNet 库。
导入其他 dotNet 库前导入此库才能修改默认运行时版本为 4.0。
注意 .NET 3.5的运行时是 v2.0，.Net 4.5 的运行时是 v4.0，
Win7 自带 .NET2.0 运行时，Win10 自带 .NET4.0 运行时。
获取系统已安装 .Net 版本信息请使用标准库 dotNet.version

dotNet.v4()

如果已成功安装 .Net 4.x 则返回 true
